All changes can be committed to this repository.
The process for the setup is : 
1) download Git for your desktop. please avoid the GUI client.
2) access the AppIdea repository, created by me.
3) look for the clone URL and copy it.
4) Open terminal/cmd and navigate to the folder to which you want to save the repository.
5)RUn the following command : -
	git clone <<URL>>
6) then run :
	git pull
